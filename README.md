Insurance CRM Portal
A full-stack insurance CRM web application built with Angular and Java Spring Boot, enabling insurance advisers to securely manage client profiles and policy information.

Tech Stack
Frontend

Angular 19
- TypeScript
- Angular Material
- SCSS

Backend

- Java 21
- Spring Boot 3.5
- Spring Security
- JWT Authentication

Database

- MySQL 9.6
- Hibernate ORM
- Spring Data JPA


Features

- JWT authentication with role-based access control
- Adviser portal to manage clients and policies
- Client profile management with full CRUD operations
- Secure REST API with Spring Security filter chain
- Responsive UI built with Angular Material
- Route guards and HTTP interceptors for secure navigation
